# SamfxBook Institutional Terminal - Public Releases

Welcome to the official public distribution repository for the **SamfxBook Institutional Terminal**.

This repository hosts the stable binaries and installation guides for our multi-platform trading terminal, ensuring a secure and authentic download environment.

## 🚀 Installation Options

### 1. Android (Direct APK)
The Android terminal provides high-performance synchronization with institutional data feeds.
*   **Download**: [SAMFXBOOK.apk (v1.0.0)](./SAMFXBOOK.apk)
*   **How to Install**:
    1. Download the `.apk` file directly to your device.
    2. Open your device **Settings** > **Security**.
    3. Enable **"Install from Unknown Sources"**.
    4. Open the downloaded file and tap **Install**.

### 2. iOS (PWA / Web-App)
Due to Apple's distribution policies, the Institutional Terminal is deployed as a secure PWA (Progressive Web App) to avoid third-party interference.
*   **Guide**: [iOS Installation Guide](https://samfxbook.vercel.app/install-guide)
*   **Direct Access**: Visit `https://samfxbook.vercel.app` in Safari and tap **"Add to Home Screen"**.

### 3. Web Dashboard
Access the full terminal from any desktop browser.
*   **URL**: `https://samfxbook.vercel.app`
